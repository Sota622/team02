# アプリ名（仮）

エコチャレンジ！ゴミ分別マスター

---

# アプリ概要

制限時間内に流れてくるゴミを、正しいゴミ箱へ分別して高得点を目指すゲームです。
ゲームを楽しみながら、ゴミの分別方法を学ぶことができます。
※ 現在はアイデア段階のため、話し合いによって変更する可能性があります。

---

# 開発目的

- 楽しみながらゴミ分別について学べるゲームを制作する。
- チームで要件定義・設計・実装まで協力して開発を行う。

---

# ゲームのルール（案）

- 制限時間内にできるだけ多くのゴミを分別する。
- ゴミを正しいゴミ箱へ入れると得点が加算される。
- 間違ったゴミ箱へ入れると減点となる。
- 制限時間終了時のスコアによって結果を表示する。
