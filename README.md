## アプリ概要

「ゴミ分別マスター」は、制限時間内に表示されるゴミを松山市のゴミ分別ルールを参考にして正しいゴミ箱へ分別し、高得点を目指すゲームです。

ゲームを楽しみながら、日常生活でも役立つゴミ分別の知識を身に付けられることを目的としています。

---

## アプリの特徴

本アプリでは、ゲームを楽しみながらゴミ分別を学べることを目的としています。

### 特徴

- 松山市のゴミ分別ルールを参考にしたゲーム
- シンプルなドラッグ＆ドロップ操作

# ゲーム仕様

## 世界観

プレイヤーは「エコチャレンジ」の参加者となり、制限時間内に松山市のゴミ分別ルールに従ってゴミを正しく分別します。
正しい分別を繰り返して高得点を獲得し、「ゴミ分別マスター」を目指します。

※世界観や設定は今後変更する可能性があります。

---

## 基本ルール

- 制限時間は60秒
- ゲーム開始後、画面下部中央にゴミが1つ表示される
- プレイヤーはマウスでゴミをドラッグし、画面上部に配置された正しいゴミ箱へ移動させる
- 正しく分別すると得点が加算され、次のゴミが表示される
- 間違ったゴミ箱へ入れると減点となる
- 制限時間終了後に結果画面へ移行する

※ゲーム時間や操作方法は変更する可能性があります。

---

## 操作方法

| 操作               | 内容               |
| ------------------ | ------------------ |
| マウス             | ゴミをドラッグする |
| ドラッグ＆ドロップ | ゴミ箱へ分別する   |
| 左クリック         | ボタンを押す       |

---

## 得点

| 行動         | 得点                      |
| ------------ | ------------------------- |
| 正しく分別   | +10点                     |
| 間違った分別 | -5点                      |
| 5回連続正解  | +20点（余力があれば実装） |

---

## ゲーム終了条件

- 制限時間（60秒）が終了するとゲーム終了
- 最終スコアを表示し、結果画面へ移行する

※終了条件は変更する可能性があります。

## ゴミ箱の種類

| ゴミ箱           | ゴミの例                         |
| ---------------- | -------------------------------- |
| 🔥 燃やせるごみ   | ティッシュ・紙くず・バナナの皮   |
| 🚯 燃やせないごみ | フライパン・乾電池・ガラス製品   |
| 🥫 缶・びん       | アルミ缶・スチール缶・ガラスびん |
| ♻️ 資源ごみ       | ペットボトル・新聞紙・雑誌       |

※松山市のゴミ分別ルールを参考にする予定です。実際のゲームで使用するゴミは話し合いを通して決定します。
